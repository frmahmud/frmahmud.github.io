Graduate Researacher Current June 24 2019 to December 11 2022

Canberra Evolutionary Optimization Research, SEIT, UNSW, Canberra, Australia

Supervisor: Professor Ruhul Sarker

Project: Robust Evolutionary Algorithms for Changing and Uncertain Environments.

Project Goal: The aim of this project is to develop an efficient algorithm to handle heterogeneous activities and create a new dataset for heterogeneous project scheduling problem (HPSP).

Contribution: I lead the project with three more contributors. As an initial contribution, I already created a new benchmark set for HPSP, as well as proposed a new variant of scheduling problem. After implementing, the experimental result of the proposed approach showed that it obtains similar quality solution with a significantly less computational time in comparison with the state-of-the-art algorithm. In future, I will consider dynamic environments such as stochastic activity durations, resource disruptions, time-varying resource demands as well as availability.
